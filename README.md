# 钉钉 AI 字段工具集

本目录包含多个基于 AI 技术的钉钉字段捷径工具，旨在提升办公效率和创意表达能力。以下是各个项目的功能和用途说明：

## 项目列表

### 1. ding-NanoAiTool - 图片编辑(Nano) 字段捷径

**功能描述**：依托最新顶尖图片模型，对话交流间便可便捷生成目标图片

**主要用途**：
- 对话式修图：以自然语言指令实现局部修改，如模糊背景、去除污渍、改变姿势、给黑白照片上色等
- 保持角色或物体外观一致：利于系列图片、产品目录和品牌素材制作
- 多图像融合：将最多三张输入图像的元素整合，创造独特构图
- 风格迁移：把一种风格应用到另一图像主体上
- 现实世界推理能力：基于 Gemini 底层逻辑，具备现实世界推理能力，增添图像编辑和生成的逻辑性与连贯性

### 2. ding-gptAiTools - AI 对话(GPT) 字段捷径

**功能描述**：用户能灵活创建自定义指令，选适配 GPT 模型执行，覆盖多样业务场景

**主要用途**：
- 灵活定制的多 GPT 模型 AI 助手：支持 gpt-5、gpt-4o-mini、gpt-5-mini、gpt-5-nano、gpt-5-thinking 等多模型选择
- 适配不同复杂度的任务：轻量任务选 gpt-4o-mini/gpt-5-nano，深度推理选 gpt-5/gpt-5-thinking
- 思路过程显示功能：操作透明可追溯，辅助用户理解 AI 决策逻辑
- 简单配置，智能执行：只需编写符合自身需求的指令，选择适配的 GPT 模型，点击执行即可

### 3. ding-grokToVideo - GrokToVideo 视频生成字段捷径

**功能描述**：文描或图传皆能生视频，自带音效，轻松出电影感成片

**主要用途**：
- 双模式创作：支持文本到视频与图像到视频双模式创作
- 多风格支持：可灵活响应写实、电影级、动漫等多种风格需求
- 极简操作流程：仅需输入文本提示或上传素材，即可一键生成完整音视频作品
- 高清流畅输出：支持生成最长 10 秒的 1080P 高清视频，部分场景可达到 720P/24 帧的电影级流畅度
- 多维音频生成：同步输出背景音效、动作声效、多语言角色对话及背景音乐

### 4. ding-sora2ToVideo - Sora2ToVideo 视频生成字段捷径

**功能描述**：文描或图传皆能生视频，自带音效，轻松出电影感成片

**主要用途**：
- 双模式创作：采用 Sora 2 驱动，支持文本到视频与图像到视频双模式创作
- 多风格支持：可灵活响应写实、电影级、动漫等多种风格需求
- 极简操作流程：仅需输入文本提示或上传素材，即可一键生成完整音视频作品
- 高清流畅输出：支持生成最长 12 秒的 1080P 高清视频，部分场景可达到 720P/24 帧的电影级流畅度
- 多维音频生成：同步输出背景音效、动作声效、多语言角色对话及背景音乐

### 5. ding-veo3ToVideo - AI 视频生成(Veo3) 字段捷径

**功能描述**：AI 生 4K 视频，简操作适配多场景提效

**主要用途**：
- 基于 Google 最新 AI 视频生成模型 Veo 3 构建
- 将文本或静态图像快速转化为电影级别的视频片段
- 自动匹配高质量音效
- 大幅降低专业视频制作门槛

## 使用说明

1. **进入对应项目目录**：每个项目都是独立的，需要进入具体的子项目目录操作
2. **安装依赖**：在子项目目录下运行 `npm install` 安装必要的依赖
3. **启动开发服务器**：在子项目目录下运行 `npm run start` 启动开发服务器
4. **构建项目**：在子项目目录下运行 `npm run build` 或 `npm run pack` 构建项目
5. **参考各项目的 README.md 文件**了解详细使用方法
6. 根据具体需求选择合适的 AI 工具

**注意**：不要在 dingdingAIfields 根目录下运行 npm 命令，应该在具体的子项目目录下运行。
